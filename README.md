# 🌱 EcoAlerta System

Sistema web para gerenciamento de ocorrências urbanas e ambientais.

## 📌 Funcionalidades

- Login e cadastro de usuários
- Controle de acesso por nível:
  - Administrador
  - Vice-líder
  - Usuário comum
- Cadastro de ocorrências
- Dashboard com gráficos
- Alteração de status de chamados
- Gerenciamento de usuários
- Integração Frontend + Backend + Banco de Dados

## 🛠 Tecnologias utilizadas

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Java Spring Boot
- Maven

### Banco de Dados
- MySQL
